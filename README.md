## Dependencies
The project uses Jest, Eslint, Prettier and Cypress
<br>

## Usage
### Installing:
```npm i```

### Build:
```npm run build```

### Unit testing:
```npm run test-unit```

### E2E testing:
```npm run test-e2e-cli```

### All tests:
```npm run test```
