## Running locally

### Install

- Clone the repo:
```
git@github.com:oBraenna91/social-media-client-workflow-CA.git
```
### Run (in terminal)
```
npm i
```
```
npm run build
```

### Testing

#### Jest
```
npm run test-unit
```
#### Cypress
```
npm run test-e2e-cli
```
#### Both Jest and Cypress
```
npm run test
```
### Format and Linting
#### To run Prettier
```
npm run format
```
#### To run ESLint
```
npm run lint
```
