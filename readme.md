## Getting Started

### Installing

* Clone Repo
* Install Dependencies
* Run build

### Executing program

* Clone repo:
```bash
git clone => branch is workflow => https://github.com/VaNomad/social-media-client-ca.git
```
* Install Dependencies:
```bash
npm i
```
* Run build:
```bash
npm run build
```
