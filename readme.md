## Getting Started

### Installing

1. Clone the repo:

```bash
gh repo clone gh repo clone robertmartinsen/social-media-client
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
